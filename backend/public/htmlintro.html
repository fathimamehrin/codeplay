This is how your HTML looks:

<!DOCTYPE html> /* at the top tells the browser, "Hey, this is an HTML page!" */

<html> /* wraps everything on the page. */

<head> /* holds hidden info like the page's title, links to styles, and metadata. */

<title>Try Me</title> /* sets the title shown in the browser tab */

</head>

<body> /* is where all the visible content goes — text, images, buttons, and more. */

<h1>Try Me!</h1>

</body>

</html>
