CSS Display

The display property is used to control how an element is shown.

<html>
<head>
        <style>
          .box {
          background-color: lightblue;
          border: 2px solid black;
          padding: 10px;
          margin: 10px;
        }
        .block {
          display: block;
        }
        .inline {
          display: inline;
        }
        .hidden {
          display: none;
        }           
      </style>
      </head>
<body>
         <h2>Block</h2>
        <div class="box block">Block element</div>
        <h2>Inline</h2>
        <div class="box inline">Inline element</div>
        <h2>None</h2>
        <div class="box hidden">Hidden element</div>
</body>
</html>
