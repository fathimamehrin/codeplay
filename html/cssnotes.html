INTRODUCTION TO CSS

CSS (Cascading Style Sheets) is used to style and design web pages. While HTML creates the structure of a website, CSS controls how it looks — colors, spacing, layout, fonts, and animations.
Think of HTML as the skeleton and CSS as the skin and clothes of a website.
To apply CSS styles, we must connect CSS with HTML. There are 3 common and beginner-friendly ways to do this.

1. INTERNAL CSS

Internal CSS is written inside the <style> tag within the <head> section of an HTML file.

<html>
<head>
<style>
p{ 
  color:blue;
  font-size:18px;
  }
  </style>
</head>
<body>

  <p>The text is styled using internal css.</p>

</body>
</html>

2. EXTERNAL CSS

External CSS is written in a seperate .css file and linked to HTML using the <link> tag.

Step 1: Create a CSS file(style.css)

p{ 
  color:blue;
  font-size:18px;
  }

Step 2: Link CSS file in HTML

<!DOCTYPE html> 
<html>
<head>
  <link rel="stylesheet"  href="style.css"
</head>
<body>

  <p>The text is styled using internal css.</p>

</body>
</html>

3. INLINE CSS

Inline CSS is written directly inside an HTML element using the style attribute.

<p style="color:Red; font-size:18px;">
 This text is styled using internal css.</p>

BASIC CSS STRUCTURE:

1. CSS Syntax

CSS is written using selectors and rules.

 selector { 
  property: value; 
 }

2. Element Selector

Selects all elements of the same type.

 p {
  color: blue;
 }

3. Class Selector

Used to style multiple elements.

 .box { 
   background-color: lightblue;
 }

4. ID Selector

Used to style a single unique element.

 #header { 
   height: 60px;
 }

CSS BOX MODEL:

1. Width & Height

Controls the size of an element.

 .box { 
   width: 200px; 
   height: 100px; 
 }

2. Padding

Space inside the element.

 .box { 
   padding: 20px;
 }

3. Margin

Space outside the element.

 .box { 
   margin: 20px; 
 }

4. Border

Outline around an element.

 .box { 
   border: 2px solid black; 
  }

5. Box Sizing

Makes sizing easier and predictable.

 * { 
   box-sizing: border-box; 
 }

COLORS & BACKGROUND:

1. Text Color

You can set the color of text

 p { 
  color: red; 
 }

2. Background Color

You can set the background color for HTML elements

 div { 
  background-color: #6a4fbf; 
 }

3. Background Image

The background-image property specifies an image to use as the background of an element.By default, the image is repeated so it covers the entire element. 

 body { 
   background-image: url("bg.jpg"); 
   background-size: cover; 
 }

TEXT & FONTS:

1. Font Family

The CSS font-family property specifies the font for an element.

 body { 
  font-family: Arial, sans-serif; 
 }

2. Font Size & Weight

The CSS font-size property is used to specify the size of the text/font. Being able to manage the text size is very important in web design.

 h1 { 
  font-size: 32px; 
  font-weight: bold; 
 }

3. Text Alignment

The text-align property is used to set the horizontal alignment of a text.

 p { 
  text-align: center; 
 }

DISPLAY & POSITION:

1. Display Property

The display property is an important CSS property for controlling layout. It specifies whether an HTML element is treated as a block or an inline element.

 span { 
  display: inline-block; 
 }

2. Position Property

CSS positioning is about controlling the placement of elements within a web page. With CSS positioning, you can override the normal document flow.

 .box { 
  position: relative; 
  top: 10px; 
 }

FLEXBOX (MODERN LAYOUT):

1. Flex Container

The flex container element can have the following properties:
display - Must be set to flex or inline-flex
flex-direction - Sets the display-direction of flex items
flex-wrap - Specifies whether the flex items should wrap or not
flex-flow - Shorthand property for flex-direction and flex-wrap
justify-content - Aligns the flex items when they do not use all available space on the main-axis (horizontally)
align-items - Aligns the flex items when they do not use all available space on the cross-axis (vertically)
align-content - Aligns the flex lines when there is extra space in the cross axis and flex items wrap

 .container { 
   display: flex; 
   justify-content: center; 
   align-items: center; 
  }

RESPONSIVE DESIGN:

1. Media Queries

CSS media queries allow you to apply styles based on the characteristics of a device or the environment displaying the web page.

 @media (max-width: 768px) { 
  .container { flex-direction: column; } 
 }

EFFECTS & ANIMATIONS:

1. Transition

CSS transitions allows you to change property values smoothly, over a given duration.

 button { 
  transition: 0.3s; 
 }

2. Transform

The CSS transform property applies a 2D or 3D transformation to an element. This property allows you to rotate, scale, move, and skew elements.

 button:hover { 
  transform: scale(1.1); 
 }

3. animation

The animation property is used to add motion effects to HTML elements using CSS. It allows elements to move, change color, size, position, or shape automatically.

3.1 @keyframes

@keyframes defines the steps of an animation — what happens at the start, middle, and end.

 @keyframes moveBox { 
    from { transform: translateX(0); } 
    to { transform: translateX(100px); } 
 }

3.2. animation-name

Specifies the name of the animation defined using @keyframes.

 .box { 
   animation-name: moveBox; 
 }

3.3. animation-duration

Defines how long the animation takes to complete one cycle.

 .box { 
   animation-duration: 2s; 
 }

3.4. animation-timing-function

Controls the speed curve of the animation.

 .box {
  animation-timing-function: ease;  
 }

BORDER PROPERTY:

1. border-width

Controls the thickness of the border.

 .box { 
   border-width: 4px; 
   border-style: solid; 
   }

2. border-style

Defines how the border looks.

 .box { 
   border-style: dashed; 
  }

3. border-color

Sets the color of the border.

 .box { 
   border-color: blue; 
  }

4. Border on Individual Sides

You can apply borders to specific sides.

 .box { 
   border-top: 2px solid red; 
   border-bottom: 2px solid blue; 
  }

5. border-radius

Makes the corners of an element rounded.

 .card { 
   border: 2px solid black; 
   border-radius: 12px; 
 }
