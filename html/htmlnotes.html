INTRODUCTION TO HTML

HTML (HyperText Markup Language) is the standard language used to create webpages. It tells the browser how to structure content — like text, images, links, etc.
Think of HTML as the skeleton of a website.

  • BASIC STRUCTURE TAGS
  • TEXT FORMATTING TAGS
  • LINKS & IMAGES
  • CONTAINERS & LAYOUT
  • LISTS
  • FORMS AND INPUT

BASIC STRUCTURE TAGS:

1. <!DOCTYPE html>

Declares that the document is using HTML5 — the latest version of HTML.

2. <html> ... </html>

Wraps the entire content of your webpage. It's the root tag of your HTML document.

<html>
  <!-- Everything else goes here -->
</html>

3. <head> ... </head>

Holds information about the page — like the title, metadata, linked CSS/JS files. This is not shown on the webpage

<head>
  <title>My First Page</title>
  <meta charset="UTF-8">
</head>

4. <title> ... </title>

Sets the title shown in the browser tab or search results.

<html>
<head>
   <title>My First Page</title>
</head>
</html>

5. <body> ... </body>

Contains all the visible content of a webpage like text, images, links, and forms.

<html>
<head>
   <title>My First Page</title>
</head>
<body>
  <p>This is visible content</p>
</body>
</html>

TEXT FORMATTING TAGS:

6. <h1> to <h6>

Used for headings. <h1> is the largest and most important, <h6> is the smallest.

<h1>Main Heading</h1>
<h2>Section Heading</h2>
<h3>Sub-Section Heading</h3>
<h4>Minor Heading</h4>
<h5>Very Small Heading</h5>
<h6>Least Important Heading</h6>

7. <p> ... </p>

Defines a paragraph of text.


<p>This is a paragraph of text</p>

8. <br>

Inserts a line break without starting a new paragraph.

                         
Hello <br> world

9. <i>, <b>, <u>

Formats text: italic, bold, and underline.

<i>italic text</p>
<b>bold text</p> 
<u>underlined text</p>

LINKS & IMAGES:

10. <a> ... </a>

Creates a clickable hyperlink.

<a href="https://example.com">
  Visit Website
</a>

11. <img>

Displays an image on the webpage.


<img src="image.jpg"  alt="Sample Image">

CONTAINER & LAYOUT:

12. <div> ... </div>

A block-level container used to group elements for layout and styling.

 <div class="box" 
 <p> Content Inside</p>
 </div>

13. <span> ... </span>

An inline container used for styling part of text.

<p>
This is <span>highlighted</span> text
</p>

LISTS:

14. <ul>

Used when the order of items is not important.

<ul>
  <li>HTML</li>
  <li>CSS</li>
  <li>JavaScript</li>
</ul>

15. <ol>

Used when the sequence of items matters.

<ol>
  <li>Open browser</li>
  <li>Write HTML</li>
  <li>Save file</li>
</ol>

16. <li>

<li> must be used inside <ul> or <ol>.

<ul>
  <!-- Each line below is a list item -->
  <li>Milk</li>
  <li>Bread</li>
</ul>

FORMS & INPUT:

17. <form> ... </form>

Used to collect user input.

<form>
  <input type="text">
  <button>Submit</button>
</form>

18. <input>

Accepts user input like text, email, password, etc.


<input type="email" placeholder="enter email"

MEDIA ELEMENTS:

19. <audio>

Embeds audio content.

<audio controls>
  <source src="music.mp3">
</audio>

20. <video>

Embeds video content.

<video controls width="300">
  <source src="video.mp4">
</video>

SCRIPTING & META TAGS:

21. <script>

Used to add JavaScript to a webpage.


<script>alert("Hello World");</script>

22. <meta>

Provides metadata like character set and page description.


<meta name="description" content="Learning HTML">
