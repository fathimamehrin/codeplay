INTRODUCTION TO JAVASCRIPT

JavaScript is a high-level, interpreted programming language used to make web pages interactive. It allows you to control behavior, respond to user actions, update content dynamically, and build full web applications.
JavaScript runs directly in the browser and works together with HTML and CSS.
To apply JS , we must connect JS with HTML. There are 3 common and beginner-friendly ways to do this.

1 INTERNAL JAVASCRIPT

Written inside the <script> tag.

<script>
alert("Hello World");
</script>

2. EXTERNAL JAVASCRIPT (Recommended)

Written in a separate .js file and linked to HTML.

<script src="script.js"></script>

3. INLINE JAVASCRIPT

Written directly inside HTML elements.

<button onclick="alert('Clicked!')">
Click Me
</button>

VARIABLES

A variable is a container used to store data values.

let age = 20;

1. var

var is the old way of declaring variables. It has function scope.

var name = "Alex";

2. let

let is used to declare variables that can change later.

let score = 50;
score = 60;

3. const

const is used for values that should not change.

const pi = 3.14;

DATA TYPES

1. String

Strings store text values.

let city = "Chennai";

2. Number

Numbers store numeric values.

let price = 199;

3. Boolean

Boolean values can be true or false.

let isLoggedIn = true;

4. typeof Operator

The typeof operator checks the data type of a value.

typeof price;

OPERATORS

1. Arithmetic Operators

Used to perform mathematical calculations.

let sum = 10 + 5;

2. Comparison Operators

Used to compare two values.

5 === 5;

3. Logical Operators

Used to combine conditions.

true && false;

CONDITIONAL STATEMENTS:

1. if Statement

Executes a block of code if the condition is true.

if (age > 18) {
  console.log("Adult");
}

2. else Statement

Runs when the if condition is false.

if (age < 18) {
  console.log("Minor");
} else {
  console.log("Adult");
}

3. else if Statement

Checks multiple conditions.

if (score >= 90) {
  grade = "A";
} else if (score >= 75) {
  grade = "B";
}

4. switch Statement

Used to perform different actions based on different values.

switch (day) {
  case 1:
    result = "Monday";
    break;
  default:
    result = "Unknown";
}

LOOPS:

1. for Loop

Repeats code a specific number of times.

for (let i = 0; i < 5; i++) {
  console.log(i);
}

2. while Loop

Runs while a condition is true.

while (count < 3) {
  count++;
}

3. do...while Loop

Executes at least once before checking condition.

do {
  count++;
} while (count < 3);

FUNCTIONS:

1. Function Declaration

Defines reusable code.

function greet() {
  console.log("Hello");
}

2. Function Parameters

Parameters are values passed to a function.

function add(a, b) {
  return a + b;
}

3. Arrow Function

Shorter syntax for writing functions.

const multiply = (a, b) => a * b;

DOM MANIPULATION:

1. Selecting Elements

Selecting elements means finding and accessing HTML elements from the webpage so JavaScript can work with them.
JavaScript uses methods like getElementById() and querySelector() to select elements.

document.querySelector(".box");

2. Changing Content

Changing content means modifying the text or HTML inside an element using JavaScript. Properties like innerText and innerHTML are used to update what appears on the webpage dynamically.

element.innerText = "Updated";

EVENTS:

1. Click Event

A click event occurs when a user clicks on an element like a button, image, or link. JavaScript listens for this action and runs code when the click happens.

btn.addEventListener("click", () => {
  alert("Clicked");
});

ARRAYS:

1. Creating an Array

Stores multiple values in one variable.

let colors = ["red", "blue"];

2. Accessing Elements

Accessing elements means getting a specific value from an array using its position

colors[0];

3. Array Methods

Array methods are built-in functions used to perform actions on arrays, such as adding, removing, or modifying items.

colors.push("green");

OBJECTS:

1. Creating an Object

Objects store data in key-value pairs.

const user = {
  name: "Alex",
  age: 20
};

2. Accessing Properties

Accessing properties means reading or using a value stored inside an object using its key name.

user.name;

LOCAL STORAGE:

1. Storing Data

Storing data means saving information in the browser so it can be used later, even after refreshing the page.

localStorage.setItem("name", "Alex");

2. Retrieving Data

Retrieving data means getting back stored information from the browser storage so it can be displayed or used again.

localStorage.getItem("name");

ERROR HANDLING:

1. try...catch

Handles runtime errors safely.

try {
  undefinedFunction();
} catch (error) {
  console.log(error);
}
